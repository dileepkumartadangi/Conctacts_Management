# Contact Manager

A lightweight contact management app built with React + Vite.  
Add, view, and delete contacts with localStorage persistence.  
This repo contains a simple project structure for beginners looking for a React project.

## Features

- Add new contacts (name, number, location)
- Delete and edit saved contacts
- Data persists in localStorage
- Simple user friendly UI

## Setup

```bash
# Clone and install
git clone https://github.com/dileepkumartadangi/Conctacts_Management.git
cd contact-manager
npm install

# Run dev server
npm run dev

# Build for production
npm run build
